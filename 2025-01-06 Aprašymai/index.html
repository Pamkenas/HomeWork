HTML

            

    <!DOCTYPE html>   - Tai yra dokumento tipo deklaracija


    <html>            - Pagrindinis HTML dokumento elementas


    <head>            - Talpina metaduomenis apie dokumentą


    <meta>            - Apibrėžia metaduomenis


    <title>           - Nustato dokumento pavadinimą


    <link>            - Susieja išorinius resursus


    <div>             - Apibrėžia skiltį ar sekciją HTML dokumente


    <h1>              - Atspindi pirmo lygio antraštę


    <p>               - Apibrėžia pastraipą

CSS

            

    Selectors  - Leidžia pasirinkti elementus pagal jų tipą, klasę, ID ir kitus atributus.


    Colors     - Nustato elementų spalvas, naudojant vardus, HEX, RGB, HSL ar kitus formatus.


    Background - Leidžia nustatyti elementų foną, pvz., spalvą, vaizdą ar gradientą.


    Font       - Valdo teksto šriftą, spalvą, dydį, stilių ir kitus parametrus.


    Margin     - Nustato išorines paraštes aplink elementą.


    Padding    - Nustato atstumą tarp elemento turinio ir jo kraštinių.


    Border     - Nustato elemento kraštinės plotį, stilių ir spalvą.


    Position   - Nustato elemento poziciją puslapyje (static, relative, absolute, fixed, sticky).


    Flexbox    - Lankstus išdėstymo modelis elementų lygiavimui ir paskirstymui konteineryje.


    Grid       - Išdėstymo sistema su eilutėmis ir stulpeliais, skirta sudėtingesniam dizainui.
