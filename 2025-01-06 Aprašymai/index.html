HTML

            
    <!DOCTYPE html>      - This is the document type declaration.
    <html>               - The root element of an HTML document.
    <head>               - Contains metadata about the document.
    </head>              - Closing the head section.
    <meta>               - Defines metadata like charset, viewport, etc.
    <title>              - Specifies the title of the document.
    <link>               - Links external resources like CSS.
    <div>                - Defines a division or section in HTML.
    <h1>                 - Represents a heading level 1.
    <p>                  - Defines a paragraph.
    <link>               - This is the Link Element for references.
